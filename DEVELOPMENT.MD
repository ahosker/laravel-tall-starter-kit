# Development Notes

## ahosker/laravel-tall-starter-kit#master

This `#master` branch is a fork of `nunomaduro/laravel-tall-starter-kit#master` with additional features and customizations for development purposes.

Master can merge from `upstream/main` with `git fetch upstream && git checkout master && git merge upstream/main`

Remember to run:

1. `composer update` # update dependencies if needed.
2. `composer lint` # run code linters.
3. `composer test` # run the test suite to ensure everything is working correctly.
4. `git push origin master` # push changes to your forked repo.

# Development Notes

## ahosker/laravel-tall-starter-kit#master

This `#master` branch is a fork of `nunomaduro/laravel-tall-starter-kit#master` with additional features and customizations for development purposes.

To follow numomaduro upstream changes run:

1. `git fetch upstream` # get latest changes from original repo.
2. `git checkout master` # switch to your local master branch.
3. `git merge upstream/main` # merge changes from original repo into your local master branch.
4. Manage any merge conflicts if they arise.
5. `composer update` # update dependencies if needed.
6. `composer lint` # run code linters.
7. `composer test` # run the test suite to ensure everything is working correctly.
8. `git push origin master` # push changes to your forked repo.
